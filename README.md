# 🎂 Hadiah Ulang Tahun Premium untuk Abang

Sebuah *Interactive Web Experience* ultra-mewah, dirancang khusus dengan teknologi web modern (HTML, CSS, JS, Three.js). Proyek ini dirancang secara spesifik dengan struktur tanpa-server sehingga 100% bisa dikelola dan di-*deploy* langsung menggunakan *smartphone* Android.

## ✨ Fitur Utama
*   **Procedural Animation:** Efek bunga pembuka menggunakan Canvas API.
*   **Three.js 3D Engine:** Animasi galaksi/partikel berbentuk hati di 60 FPS.
*   **Mic-Sensor Candle:** Teknologi Web Audio API untuk deteksi tiupan nafas.
*   **Mobile-First Glassmorphism UI:** Estetika premium ala *Apple* dan *Pinterest*.
